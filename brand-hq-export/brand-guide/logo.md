# Logo Usage

The Vinyl G mark is the core identifier -- a concentric spiral inspired by vinyl grooves. It works alone, with the wordmark, or in specialty lockups. Always use approved files -- never recreate or approximate.

**Asset downloads:** All logo types available in SVG, PNG, JPEG, and PDF from the `Logo Assets/` folder and Google Drive.


## Primary Logo

The **Horizontal Lockup** is the primary logo. Use it wherever space allows -- nav bars, headers, footers, presentations, and documents. Default to Warm Bronze on dark backgrounds.

File: `Logo Assets/SVG/Grouped_Rebrand_Visual Assets_Lockup Horizontal_Warm Bronze.svg`


## Core Logo System

Three core components form the logo system:

| Component | Description | Use For |
|---|---|---|
| **Vinyl G Icon** | Icon mark alone | Favicons, app icons, avatars, tight spaces |
| **Lockup Horizontal** (PRIMARY) | Icon + wordmark side by side | Nav bars, headers, footers, presentations |
| **Lockup Vertical** | Icon stacked above wordmark | Social profiles, splash screens, square formats |

## Clear Space

Minimum clear space around the logo equals the height of the Vinyl G mark (1x the mark). No other elements, text, or edges may intrude into this zone.

**Rule:** 1x mark height on all sides.


## Minimum Size

| Logo Type | Minimum | Context |
|---|---|---|
| **Icon only** | 24px | Favicons, app icons, small UI |
| **Horizontal lockup** | 100px wide | Nav bars, email signatures, footers |
| **Vertical lockup** | 64px tall | Social profiles, splash screens |

Below these thresholds, detail degrades. Use the icon alone at the smallest sizes -- never scale the horizontal lockup below its minimum.


## Do

- Always use approved logo files from the asset library
- Maintain clear space of at least 1x the mark height
- Use the Vinyl G icon alone when space is tight (favicons, app icons)
- Use Warm Bronze as the default color on dark backgrounds
- Use the Horizontal Lockup for nav bars, headers, and footers
- Use the Vertical Lockup for social profiles and splash screens
- Keep the wordmark in Anacrusis -- never substitute Satoshi


## Don't

- Don't recreate or redraw the Vinyl G -- always use vector source
- Don't rotate, skew, or apply perspective transforms
- Don't place the logo on busy or low-contrast backgrounds
- Don't change the proportions of the concentric arcs
- Don't use gradients, shadows, or effects on the mark
- Don't separate elements within any lockup
- Don't lock up the mark with taglines or other icons

## Cross-Surface Translation

| Surface | How Logo Shows Up |
|---|---|
| **Product (app)** | Vinyl G icon in nav sidebar (24-32px), horizontal lockup on login/splash |
| **Marketing site** | Horizontal lockup in nav bar (Warm Bronze), vertical lockup on hero sections |
| **Mobile** | Vinyl G icon for app icon and in-app nav, horizontal lockup on splash screen |
| **Print** | Vector source files only, Pantone spot colors for premium print |
| **Events/experiential** | Large-format vertical lockup on backdrops, Vinyl G icon on badges/wristbands |
| **Social** | Vertical lockup for profile images, Pill Tag or Retro Stamp for special posts |
| **Email** | Horizontal lockup in email header (PNG @2x fallback), min 100px wide |
| **Investor materials** | Horizontal lockup on title slides, Vinyl G icon as deck watermark |
