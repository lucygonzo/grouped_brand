# Grouped Brand Quick-Start

Everything you need in 30 seconds. Drill into the deep reference sections when you need specifics.


## Four Core Colors

| Color | Hex (Dark) | Hex (Light) | Role |
|---|---|---|---|
| **Bronze** | `#C48A3A` | `#956A1D` | Brand accent — logos, headlines, CTAs |
| **Navy** | `#111620` | n/a | Foundation — backgrounds, surfaces |
| **Cream** | `#F0EBE3` | n/a | Body text (dark mode), backgrounds (light mode) |
| **Focus Blue** | `#7CBBDF` | `#2874A6` | Interactive — focus rings, active states, badges |

**Rule:** Bronze = brand voice. Focus Blue = interface. Never mix them.


## Two Fonts

| Font | Role | Weights |
|---|---|---|
| **Anacrusis** | Display headlines (32px+ marketing, 40px+ product). Max 2 per screen. | Single weight |
| **Satoshi** | Everything else — body, UI, buttons, nav, headings, captions | 300, 400, 500, 700, 900 |

No Semibold (600) — use Bold (700) instead.


## Logo

The "grouped." wordmark in Anacrusis. Always includes the bronze dot.

9 variants available: Horizontal / Vertical / VinylG Icon × Warm Bronze / Cream / Black.

SVGs in Google Drive > Visual Identity Assets.


## Three Surface Types

| Surface | Feel | When to Use |
|---|---|---|
| **Solid** | Structural, grounded | Default — cards, containers, nav, panels |
| **Glass** | Frosted, floating | Overlays, callouts, popovers, modals |
| **Vinyl** | Premium, warm sheen | Hero moments, featured content (max 1–2/page) |


## Three Themes

| Theme | Base | When |
|---|---|---|
| **Dark (default)** | Navy `#111620` | Primary experience, always |
| **Light** | Cream `#FAFAF8` | User preference, product only |
| **OLED** | True black `#000000` | Mobile battery savings |


## Quick Don'ts

- Don't use neutral greys in dark mode — always navy blues
- Don't use Anacrusis under 32px or for body text
- Don't use bronze for interactive feedback (focus rings, status badges)
- Don't use gradient fills on product surfaces — solid L0–L5 tokens only
- Don't put bronze on non-navy dark backgrounds (turns muddy brown-green)
